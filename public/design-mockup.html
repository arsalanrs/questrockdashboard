↑ What's different vs. current — Design improvements proposed
📐
Bento grid layout
Variable tile sizes by importance. Hero KPIs dominate the top. Supporting cards fill in below. Nothing is the same size just to fill space.
LAYOUT
🎨
Semantic color coding
Red = urgent action. Amber = watch. Green = healthy. Blue = info. Yellow = primary metric. Every color has one meaning — never decorative.
COLOR
✦
Geist font + proper type scale
Current: everything is 11–13px. Proposed: KPI values at 30px bold, titles at 21px, body at 13px, labels at 11px. Breathing room between tiers.
TYPOGRAPHY
⚡
Tabs replace giant tables
Stuck / SLA Breach / No Contact are now tabs inside one card — less scrolling, same data. The count badge tells you immediately where to look.
DENSITY
📊
Sparkline 7-day trend
Each LO row gets a tiny bar chart showing activity trend. Falling trend = yellow flag. Immediately visible without reading numbers.
DATA VIZ
🔍
LO filter chips
Filter the "What's Not Moving" list by individual LO with one click. No page reload. Scoped to the right person instantly.
INTERACTION
📏
2px top-border color coding on KPIs
The thin colored bar at the top of each KPI card signals urgency at a glance — no need to read the label to know if something is red.
VISUAL CUE
💡
MTD / QTD / YTD toggle
All KPIs update together for the selected window. Currently the dashboard has no time-window control — you can't compare periods.
FILTER
🏥
SLA health bars
Stage-by-stage SLA health in a compact card. Which stage is your biggest bottleneck? You see it in one glance — no table needed.
